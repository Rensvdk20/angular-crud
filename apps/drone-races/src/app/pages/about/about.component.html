Drone Races

Casus:

Deze casus gaat over drone races. Deze worden georganiseerd op een specifieke datum en drone racers kunnen zich hiervoor inschrijven.
Ook kunnen mensen zich registreren op de website en na het inloggen kunnen ze tickets bestellen om naar de match te gaan.
Een gebruiker kan matches aangeraden krijgen op basis van een droneracer die heeft geracet bij een match waar de gebruiker eerder is geweest.

Het is ook mogelijk te registreren als drone racer en je in te schrijven voor een match die voldoet aan je eigen ranking.
De drone die je wilt gebruiken voor de race kan je ook beheren.

Entiteiten:

User:

Info
  • Een user kan 1 of meerdere tickets bestellen voor een match.
  • Een user kan een drone racer zijn.
  • Een user kan tot 7 dagen van te voren zijn/haar ticket cancellen.
Attributen
  • Voornaam
  • Achternaam
  • Email
  • Geboorte datum
  • Is een admin
  • Drone racer

Drone Racer:

Info
  • Een drone racer is een user maar een user hoeft niet een drone racer te zijn.
  • Als drone racer kan je jezelf inschrijven voor een match als je voldoet aan de benodigde ranking.
Attributen
  • Team
  • Ranking
  • Drone

Drone:

Info
  • Een drone heeft een naam, categorie (quadcopter, hexacopter), gewicht en of het brushless motors heeft of niet
  • Een drone kan maar bij 1 drone racer horen.
Attributen
  • Naam
  • Categorie
  • Gewicht
  • Heeft brushless motors

Ticket:

Info
  • Type kan bijvoorbeeld VIP of normaal zijn
  • Een ticket hoort bij 1 match en kan gekoppeld zijn aan 1 user.
Attributen
  • Prijs
  • Type
  • Stoelnummer
  • Match
  • Gereserveerd door

Match:

Info
  • De winnaar is gekoppeld aan de racer die de match heeft gewonnen
  • De rank is er zodat niet elke racer kan meedoen aan elke race
Attributen
  • Naam
  • Datum
  • Locatie
  • Rank
  • Prijzengeld
  • Winnaar
